Project Overview

Mee Maid is a course project showcasing UI design concepts for a mobile application that connects homeowners with professional cleaning services. This project demonstrates various app screen designs and user interface elements, focusing on creating intuitive and visually appealing layouts for a cleaning service platform. The design process involved creating comprehensive UI examples using modern design principles and mobile-first approach.

My Role

In this course project, I collaborated with my team as the primary UI designer. My responsibilities included designing the visual interface elements, creating consistent design systems, and ensuring all screen layouts met usability standards while working closely with team members to align with project requirements.

UI Design

Created clean and modern interface layouts, color schemes, and visual elements for various app screens.

Screen Layouts

Designed comprehensive UI examples including navigation, forms, and interactive elements across different app sections.

Team Coordination

Collaborated with team members to ensure design alignment and consistency throughout the project deliverables.

UI Components Designed

  • Onboarding and welcome screens with engaging visuals
  • User registration and login interface designs
  • Service booking screens with calendar and time selection
  • Service provider profile layouts and rating displays
  • Payment interface mockups and confirmation screens
  • Notification and messaging interface examples
  • User dashboard and booking history layouts

Design Challenges

The main challenge was creating UI designs that would feel trustworthy and professional for a service that involves home access. Our team focused on developing interface elements that would convey security, reliability, and ease of use through careful visual design choices and layout decisions.

Design Solutions

Our team delivered a comprehensive set of UI examples that demonstrate a cohesive design system. The interfaces prioritize clarity, visual hierarchy, and user-friendly interactions suitable for a cleaning service application.

  • Clean and intuitive interface layouts with clear navigation paths
  • Consistent visual design system with appropriate color psychology
  • User-friendly form designs for booking and profile management
  • Professional service provider presentation layouts
  • Trust-building elements through design choices and visual cues
