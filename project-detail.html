Mee Maid App

Mobile application designed to connect homeowners with professional cleaning services

2024
UI/UX, Mobile App
Personal Project

Project Overview

Mee Maid is a mobile app prototype designed to connect homeowners with professional cleaning services. The goal of this project was to create an intuitive and user-friendly experience for booking home cleaning services, from initial search to final payment. The design process involved user research, wireframing, and creating a high-fidelity prototype using Figma.

My Role

As the sole designer on this project, I was responsible for the entire UI/UX design process. This included conducting market research, defining user personas, creating user flows, designing wireframes and high-fidelity mockups, and developing an interactive prototype.

UI Design

Created a clean, modern, and engaging visual interface with a clear hierarchy.

UX Research

Conducted user interviews and competitive analysis to understand user needs and pain points.

Prototyping

Built an interactive prototype in Figma to demonstrate the app's functionality and user journey.

Key Features

  • User-friendly booking interface with calendar integration
  • Service provider profiles with ratings and reviews
  • Secure payment flow design
  • Push notification system for booking updates
  • In-app communication interface between user and provider
  • Service history and booking management

The Challenge

The main challenge was to design a seamless and trustworthy booking experience for users who are entrusting strangers with access to their homes. I addressed this by focusing on clear communication, transparent pricing, and a robust rating/review system to build confidence.

The Solution

The final design offers a simple, step-by-step booking process. Users can easily customize their service, view detailed provider profiles, and track the status of their booking. The design prioritizes clarity and security to ensure a positive user experience.

  • A visually appealing and easy-to-navigate booking flow.
  • Detailed and verified profiles for all service providers.
  • Integration of a secure payment gateway for peace of mind.
  • Real-time tracking of service status and provider location.
